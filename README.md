SPHinXsys是一个强大的开源C++库，用于基于光滑粒子流体动力学（SPH）方法的多物理场仿真。它广泛应用于计算流体力学、固体力学和流固耦合等领域。SPHinXsys提供了丰富的功能和模块，支持高效的数值计算和复杂的物理现象模拟。
